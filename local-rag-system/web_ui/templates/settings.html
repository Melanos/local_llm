Tuning Guidelines
Relevance Thresholds
  • 0.1-0.2: Very broad search (lots of results)
  • 0.3-0.4: Balanced (current setting)
  • 0.5-0.7: Precise search (fewer results)
  • 0.8+: Very strict (may miss relevant content)
Text Chunking
  • Smaller chunks (500-800): More precise search
  • Larger chunks (1000-1500): More context
  • Overlap (100-300): Prevents information loss
